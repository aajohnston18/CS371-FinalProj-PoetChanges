Introduction to Accessible Images

As the publishing world shifts to electronic media, we have an unprecedented opportunity to ensure all content that is "born digital" is also "born accessible" so that all readers have access to the information they need. In many cases, a simple text description of an image will provide all the necessary information, but determining how much text is sufficient depends on the audience and context of the image.


Use this Poet Image Description Training tool to:
  • Review a collection of sample images to determine when a description is useful.
  • Review general guidelines and practice choosing the most appropriate description.
  • Practice writing descriptions for a variety of image types.
Poet is open source and developed for the community. Your feedback helps to make Poet even better!
WHEN To Describe Images
Does an image need to be described? Well, it depends...
  • How important is the for understanding the core content?
  • What is the purpose of the image? Is it "eye candy" or associated with a function (like an icon) ?
  • Is the image already adequately described in the surrounding text?
  • Is it better to provide a tactile graphic rendering of the image?
