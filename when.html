WHEN To Describe Images

Does an image need to be described? That depends... learn about when and how best to make an image accessible.
  • How important is the image for understanding the core content?
  • What is the purpose of the image - is it "eye candy" (as in this sunset) or associated with a function (like an icon)?
  • Is the image already adequately described in the surrounding text?
  • Is it better to provide a tactile graphic rendering of the image?
